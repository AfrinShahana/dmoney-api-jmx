
#  dmoney-Api-jmx

## How to Generate load test report from the CLI
-download .jmx file 
-keep this in \apache-jmeter-5.5\bin
-open this with cmd 
-jmeter -n -t [jmx file] -l [results file] -e -o [Path to web report folder]
